# 🐰 Rabbit - AI Powered Language Assistant

Rabbit is a Flask-based web application designed to break language barriers and improve communication through multiple AI-powered language tools. The platform provides translation, dictionary lookup, grammar checking, text summarization, vocabulary building, and an interactive chatbot in a single user-friendly interface.

## 🚀 Features

- 🌍 **Language Translation**
  - Translate text between multiple languages.
  - Fast and accurate translations.
- 📖 **Smart Dictionary**
  - Search word meanings, synonyms, antonyms, and usage examples.
- ✍️ **Grammar & Spell Checker**
  - Detect grammatical mistakes and spelling errors.
  - Suggest corrections for improved writing.
- 📝 **Text Summarizer**
  - Generate concise summaries from lengthy text content.
- 📚 **Vocabulary Builder**
  - Learn new words and improve vocabulary through interactive learning.
- 🤖 **AI Chatbot**
  - Conversational assistant for language-related queries and learning support.
- 🎨 **Responsive User Interface**
  - Clean and modern design built using HTML, CSS, and JavaScript.

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Munna727/Rabbit.git
```

2. Navigate to the project directory

```bash
cd Rabbit
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```text
http://127.0.0.1:5000
```

## 🎯 Project Objective

The objective of Rabbit is to provide a centralized platform where users can access multiple language assistance tools without switching between different applications. It helps students, professionals, and language learners improve communication efficiency and productivity.
